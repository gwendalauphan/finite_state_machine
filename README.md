## Build and Run the project

The project uses CMake for the compilation. The Build is described in the file `CMakeLists.txt`


### Installation of CMake

To install CMake on Ubuntu you need to run the following commands:

```bash
sudo apt update
sudo apt install cmake
```

### Build of the project
```bash
cd finite_state_machine
cmake .
make
```

### Run
```bash
./automate
```
